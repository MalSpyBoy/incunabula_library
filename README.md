Incunabula Library

A ~28GB plain-text (+ links to much more) library with a small [~10.8GB] footprint and a PySide6 GUI

Powered by the dataset "75,000+ Gutenberg Books and Metadata 2025" on Kaggle, which can be accessed at:
https://www.kaggle.com/datasets/lokeshparab/gutenberg-books-and-metadata-2025

The dataset was built with the Gutenburg python library, so if you wish to make your own dataset for it, there are some initial directions on the linked Kaggle site.

Directions:
 - Ensure that a .zip of the above dataset, with the name 'archive.zip' is present in the directory
 - Ensure that you have PySide6 installed, which can be installed with pip: 'pip install PySide6' or 'pip install -r requirements.txt'
 - Run main.py
